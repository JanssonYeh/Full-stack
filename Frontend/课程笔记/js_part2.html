2. 回顾上节课内容

变量类型：讲解了 var 和 let 的区别，尤其是作用域与变量提升的不同。

作业：作业内容是如何不使用第三个变量来交换两个变量的值。通过加减运算可以实现变量的交换。

代码示例：

let p = 5;
let q = 10;
p = p + q; // p = 15
q = p - q; // q = 5
p = p - q; // p = 10

console.log(`p = ${p}, q = ${q}`);  // 输出 p = 10, q = 5

3. 数字与字符串的类型转换

讨论了数字与字符串的转换，特别是隐式类型转换在加法和减法中的不同处理。

加法：字符串与数字相加时，结果是字符串拼接。

减法：字符串与数字相减时，字符串会隐式转换为数字。

代码示例：

let result = '1' + 2;  // 输出 '12'，字符串拼接
let result2 = '5' - 2; // 输出 3，数字运算

4. 布尔运算与条件判断

等号比较：

  • ==：只比较值，不比较类型。
  • ===：同时比较值和类型。

代码示例：

let x = 5;
let y = "5";

console.log(x == y);  // 输出：true
console.log(x === y); // 输出：false

5. 循环与分支

讲解了如何使用嵌套循环实现九九乘法表，并解释了 continue 和 break 的使用。

九九乘法表代码示例：

for (let i = 1; i <= 9; i++) {
  for (let j = 1; j <= i; j++) {
    console.log(`${i} * ${j} = ${i * j}`);
  }
}

6. 逻辑运算与布尔值

讨论了如何进行真假值转换，以及如何使用两个 ! 来强制转换布尔值。

代码示例：

let x = 0;
console.log(!!x);  // 输出 false

let y = 1;
console.log(!!y);  // 输出 true

7. 数组（Array）操作

讲解了数组的基础操作，如获取数组的长度和使用索引访问元素。

代码示例：

let array = [1, 2, 3, "hello", true];
console.log(array[0]);  // 输出 1

8. 复杂类型与浅复制

复杂类型（如数组和对象）的复制是引用传递而不是值传递，因此改变副本中的内容会影响到原数组。

代码示例：

let array1 = [1, 2, 3];
let array2 = array1;
array2[0] = 100;
console.log(array1[0]);  // 输出 100

9. 栈与队列

栈：后进先出（LIFO），使用 push() 和 pop() 操作。

队列：先进先出（FIFO），使用 unshift() 和 shift() 操作。

代码示例：

// 栈操作
let stack = [];
stack.push(1);
stack.push(2);
console.log(stack.pop());  // 输出 2

// 队列操作
let queue = [];
queue.unshift(1);
queue.unshift(2);
console.log(queue.shift());  // 输出 2
