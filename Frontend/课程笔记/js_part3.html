JavaScript 课程笔记

1. 上节课回顾 - 复杂类型：数组（Array）

数组 是 JavaScript 中的第一个复杂类型。它的定义和使用是通过中括号 [] 来创建，数组内部可以存储任何类型的元素，包括字符串、数字、布尔值，甚至其他数组或对象。

数组的特点：

  • 数组的长度可以通过 .length 属性读取和修改。
  • 可以使用 .push() 方法向数组末尾添加新元素，使用 .pop() 从数组末尾删除元素。
let arr = [1, 2, 3];
console.log(arr.length);  // 输出 3

arr.push(4);  // 向数组末尾添加元素
console.log(arr);  // 输出 [1, 2, 3, 4]

arr.pop();  // 移除数组末尾元素
console.log(arr);  // 输出 [1, 2, 3]

特殊情况：

如果你向数组中添加的不是数字（如字符串或负数），这些元素可能不会计入 .length 计算，特别是在特定的上下文中。

2. 数据结构：栈与队列

栈（Stack）和队列（Queue）是 JavaScript 中两种重要的数据结构：

  • 栈（Stack）：后进先出（LIFO），可以通过 .push() 向栈顶添加元素，通过 .pop() 从栈顶移除元素。
  • 队列（Queue）：先进先出（FIFO），可以通过 .unshift() 向队列头部添加元素，通过 .shift() 从队列头部移除元素。
// 栈的操作
let stack = [];
stack.push(1);
stack.push(2);
console.log(stack.pop());  // 输出 2

// 队列的操作
let queue = [];
queue.unshift(1);
queue.unshift(2);
console.log(queue.shift());  // 输出 2

3. 对象（Object）的基本结构

对象（Object） 是由键值对组成的。对象的属性可以通过 点语法 或 中括号语法 来访问。

const person = {
  name: "Tom",
  age: 18
};

// 访问属性
console.log(person.name);  // 输出 "Tom"

// 修改属性
person.name = "Jerry";
console.log(person.name);  // 输出 "Jerry"

// 新增属性
person.gender = "male";
console.log(person.gender);  // 输出 "male"

4. 对象与数组的区别

虽然对象和数组都属于复杂类型，但它们的使用场景不同：

  • 数组 是有序的，可以通过数字索引访问元素。
  • 对象 是无序的，键可以是字符串，常用于表示一组有命名的属性。
// 数组
let arr = [10, 20, 30];
console.log(arr[0]);  // 输出 10

// 对象
let obj = { first: "Apple", second: "Banana" };
console.log(obj["first"]);  // 输出 "Apple"

5. 浅拷贝与深拷贝

在 JavaScript 中，拷贝对象分为浅拷贝和深拷贝：

  • 浅拷贝：只复制对象的第一层属性，复杂类型（如数组和对象）只复制引用。
  • 深拷贝：复制对象的所有层次，创建对象的全新副本。
// 浅拷贝
let arr1 = [1, 2, 3];
let arr2 = [...arr1];
arr2[0] = 100;
console.log(arr1[0]);  // 输出 1
console.log(arr2[0]);  // 输出 100

// 深拷贝
let obj1 = { name: "Tom", friends: ["Alice", "Bob"] };
let obj2 = JSON.parse(JSON.stringify(obj1));  // 深拷贝
obj2.friends.push("Charlie");
console.log(obj1.friends);  // 输出 ["Alice", "Bob"]
console.log(obj2.friends);  // 输出 ["Alice", "Bob", "Charlie"]

6. JSON 与对象的关系

JSON（JavaScript Object Notation） 是一种轻量级的数据交换格式。JSON 键和值都必须用双引号包裹，且不能包含函数。

{
  "name": "Tom",
  "age": 18
}

7. 函数基础与箭头函数

函数 是 JavaScript 中的基础构造块。我们可以使用函数来封装逻辑代码。

// 普通函数
function add(a, b) {
  return a + b;
}
console.log(add(2, 3));  // 输出 5

// 箭头函数
const addArrow = (a, b) => a + b;
console.log(addArrow(2, 3));  // 输出 5

8. 函数式编程与柯里化

柯里化（Currying） 是将多个参数的函数转换为一系列接受单个参数的函数的技术。

const multiply = a => b => c => a * b * c;
console.log(multiply(2)(3)(4));  // 输出 24
