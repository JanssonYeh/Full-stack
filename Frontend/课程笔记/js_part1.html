2. Console 基础操作

console.clear(): 清空控制台的输出。

console.clear();

console.log(): 打印信息到控制台。可以用 console.log() 在控制台输出自定义信息或变量。

console.log('====== Start ======');
console.log('====== End ======');

3. 变量和作用域

使用 let 声明变量： let 声明的变量只能在其所在的块中使用，并且可以修改值。

let a = 1;
console.log(a);  // 输出: 1
a = 2;
console.log(a);  // 输出: 2

使用 var 声明变量： var 声明的变量可以在同一个作用域内重新声明，存在变量提升（hoisting）问题。

var b = 1;
console.log(b);  // 输出: 1
var b = 2;       // 允许重新声明
console.log(b);  // 输出: 2

4. 交换变量的值

使用第三个变量来交换两个变量的值：

let p = 5;
let q = 6;

let temp = p;
p = q;
q = temp;

console.log(p);  // 输出: 6
console.log(q);  // 输出: 5

5. 变量提升（Hoisting）

使用 var 声明的变量会进行提升，而 let 和 const 不会。

console.log(a);  // 输出: undefined
var a = 5;

console.log(b);  // 会报错，因为 `let` 不会提升
let b = 10;

6. 常量和变量

const 声明常量： const 定义的常量不能修改，但可以在初始化时赋值其他变量的值：

const a = 10;
// a = 20;   // 报错：常量不能重新赋值

7. 隐式类型转换

加法运算： 如果字符串和数字相加，结果会转换为字符串：

let result = '1' + 2;
console.log(result);  // 输出: '12' (字符串)

减法运算： 如果数字和字符串进行减法，字符串会转换为数字进行计算：

let result = '5' - 2;
console.log(result);  // 输出: 3 (数字)

8. 布尔值与判断

布尔值： true 和 false 是布尔类型，用于逻辑判断。

let isAdult = true;
if (isAdult) {
  console.log('You are an adult');
} else {
  console.log('You are not an adult');
}

双等号（==）和三等号（===）：

console.log(1 == '1');  // 输出: true
console.log(1 === '1');  // 输出: false

9. 循环结构

while 循环： while 循环会一直执行，直到条件为 false。

let i = 1;
let q = 10;
while (i <= q) {
  console.log(i);
  i++;
}

for 循环： for 循环更简洁，可以将初始化、条件和增量操作合并在一行：

for (let i = 1; i <= 10; i++) {
  console.log(i);
}

9. 九九乘法表

使用 for 循环嵌套输出九九乘法表：

for (let i = 1; i <= 9; i++) {
  for (let j = 1; j <= i; j++) {
    console.log(`${i} * ${j} = ${i * j}`);
  }
}

10. DOM 操作

修改网页内容，通过 JavaScript 可以操作 DOM 元素，将内容写入网页：

<div id="app"></div>

使用 JavaScript 修改 app 元素的内容：

const message = 'Hello World';
const domElement = document.querySelector('#app');
domElement.textContent = message;
