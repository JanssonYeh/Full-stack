Frontend Setup Guide with Vite and Sass

一、安装 Node.js 和 Volta

1. 安装 Node.js

你可以从 Node.js 官方网站 下载 Node.js，并安装 LTS（Long Term Support，长期支持）版本。

2. 安装 Volta

Volta 是一个用于管理 Node.js 版本的工具，可以帮助你轻松地在不同项目中使用不同的 Node.js 版本。

安装 Volta（macOS 和 WSL）：打开终端并执行以下命令：

curl https://get.volta.sh | bash

安装 Volta（Windows）：

  1. 下载 Volta 的 Windows 安装包。
  2. 启用 Windows 开发者模式（路径：Update & Security -> For Developers -> Developer Mode）。

使用 Volta 安装 Node.js：

安装 Volta 后，你可以使用以下命令安装 Node.js：

volta install node

二、NPM 和 Yarn 的基本操作

1. 安装 NPM 和 Yarn

NPM 是 Node.js 自带的包管理工具，无需额外安装。

Yarn 是 Facebook 开发的另一个包管理工具，可以通过 NPM 安装：

npm install -g yarn

或者使用 Volta 安装 Yarn 的特定版本：

volta install yarn@1.22.22

2. NPM 和 Yarn 的常用命令

安装依赖：

使用 NPM：

npm install <package_name>

使用 Yarn：

yarn add <package_name>

安装开发依赖：

使用 NPM：

npm install <package_name> --save-dev

使用 Yarn：

yarn add <package_name> --dev

运行脚本：

使用 NPM：

npm run <script>

使用 Yarn：

yarn <script>

三、使用 Vite 构建前端项目

1. 使用 Vite 创建项目

Vite 是一个快速的前端构建工具。你可以使用 NPM 或 Yarn 来初始化 Vite 项目。

使用 NPM 创建 Vite 项目：

npm create vite@latest

在命令执行后，根据提示选择项目模板（如 Vanilla、React、Vue 等），并输入项目名称。

使用 Yarn 创建 Vite 项目：

yarn create vite

2. 启动开发服务器

在 Vite 项目目录下，运行以下命令来启动开发服务器：

使用 NPM：

npm run dev

使用 Yarn：

yarn dev

服务器启动后，Vite 会在本地 localhost 上提供开发服务器，通常是 http://localhost:5173。

3. Vite 配置文件

Vite 使用 vite.config.js 进行项目配置。以下是一个简单的 Vite 配置文件示例：

import { defineConfig } from 'vite';

export default defineConfig({
  // 配置选项
});

四、使用 Sass 和 SCSS

1. Sass 和 SCSS 的基本概念

Sass 是一种增强的 CSS 预处理语言，允许使用变量、嵌套规则、混合等高级功能。
SCSS 是 Sass 的一种语法，完全兼容 CSS。

2. 在项目中使用 Sass

你可以通过以下命令安装 Sass：

使用 NPM：

npm install sass --save-dev

使用 Yarn：

yarn add sass --dev

安装后，你可以在项目中使用 .scss 文件来编写样式。例如：

创建一个 SCSS 文件（src/styles/main.scss）：

$primary-color: #3498db;

body {
  font-family: Arial, sans-serif;
  background-color: $primary-color;
}

在 JavaScript 中导入 SCSS 文件（src/main.js）：

import './styles/main.scss';

3. Sass 的基本用法

你可以使用以下语法来增强 CSS 开发：

变量：

$primary-color: #333;

.header {
  color: $primary-color;
}

嵌套规则：

.nav {
  ul {
    margin: 0;
    padding: 0;
    list-style: none;
  }

  li {
    display: inline-block;
    margin-right: 5px;
  }
}

五、项目构建及打包

1. 构建项目

构建项目可以生成生产环境所需的静态文件，通常用于上线发布。

使用 NPM：

npm run build

使用 Yarn：

yarn build

2. 预览构建结果

构建完成后，你可以在本地预览生成的生产环境文件：

使用 NPM：

npm run preview

使用 Yarn：

yarn preview
