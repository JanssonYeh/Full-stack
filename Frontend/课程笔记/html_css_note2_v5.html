CSS 课程详细笔记

1. CSS 基础写法和选择器

  • CSS 基础写法:

    在 CSS 中，我们使用选择器来指定 HTML 元素，应用样式规则。每个样式规则包含一个选择器和一组声明（属性和值）。

    选择器示例：

    html, body { margin: 0; padding: 0; }

    这是一个组合选择器（使用逗号 "," 分隔）。它同时选中了 html 和 body，并应用相同的 margin 和 padding 样式。

    * { margin: 0; padding: 0; }

    通配符选择器 * 选择所有元素，常用于重置浏览器默认的边距和填充。

  • :root 伪类:

    :root 是一个特殊的选择器，它选中文档的根元素，在 HTML 中，它对应 <html> 元素。使用 :root 可以定义全局的 CSS 变量，供整个文档使用。

    变量定义：

    :root {
  --color-A: #e8c094;
  --color-B: #c0cf8c;
  font-size: 10px;
}

    变量引用：

    .element {
  color: var(--color-A);
}

    优势: 使用 CSS 变量可以简化样式管理，尤其在需要多次使用相同样式值时，修改方便快捷。

2. 全局样式重置与基础样式设置

  • 全局样式重置:

    为了消除浏览器之间默认样式的差异，通常会使用全局样式重置。

    示例：

    * {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

    解释：

    • margin: 0; 和 padding: 0; 用于移除所有元素的默认外边距和内边距。
    • box-sizing: border-box; 改变盒模型计算方式，使得元素的 width 和 height 包含了 padding 和 border，不需要手动计算。
  • 基础样式设置:

    通过设置 font-size 等属性，可以建立一个基础的视觉样式。

    例如：

    :root {
  font-size: 10px; /* 设置基础字体大小为10px */
}

body {
  font-size: 1.6rem; /* 使用 rem 单位，相对于 :root 的字体大小 */
}

    使用 rem 单位的好处是可以根据根元素的字体大小进行相对缩放，适应不同的设备和屏幕分辨率。

3. 盒模型与 box-sizing 属性

  • 盒模型:

    每个 HTML 元素的盒模型包含内容（content）、内边距（padding）、边框（border）和外边距（margin）。

    默认情况下，CSS 的盒模型只包含内容的宽度和高度，不包含内边距和边框。这使得在调整元素大小时需要额外计算。

  • box-sizing 属性:

    box-sizing: content-box;（默认）：宽度和高度仅包含内容，不包含 padding 和 border。

    box-sizing: border-box;：宽度和高度包含 padding 和 border，不需要额外计算。

    使用示例：

    * {
  box-sizing: border-box;
}

4. CSS 定位属性详解

  • 定位属性:

    static（默认）：元素按照正常文档流进行排列。

    relative：元素相对于其正常位置进行偏移，但仍占据原位置。

    absolute：元素相对于最近的已定位祖先元素进行定位，并脱离文档流，不再占据原位置。

    fixed：元素相对于浏览器窗口进行定位，不随滚动条滚动。

    sticky：元素在跨越特定阈值后会“粘住”在某个位置，直到超出父元素的边界。

  • 详细说明:

    relative 定位:

    相对定位允许我们通过 top、right、bottom、left 属性来调整元素相对于其正常位置的偏移量。

    使用场景：需要在正常文档流中保留元素原始位置，但希望元素有微小移动。

    .relative {
  position: relative;
  top: 10px;
  left: 5px;
}

    absolute 定位:

    绝对定位使元素脱离文档流，定位相对于最近的已定位祖先（即不是 static 的祖先元素）。如果没有已定位祖先，则相对于 <html> 元素。

    使用场景：弹出层、模态框等需要精确控制位置的元素。

    .absolute {
  position: absolute;
  top: 20px;
  right: 30px;
}

    fixed 定位:

    固定定位相对于浏览器窗口，不会随页面滚动条滚动。

    使用场景：固定导航栏、侧边栏。

    .fixed {
  position: fixed;
  top: 0;
  left: 0;
}

    sticky 定位:

    粘性定位结合了相对定位和固定定位的特点。元素在页面滚动到一定位置时变成固定定位。

    使用场景：表头固定、广告栏等。

    .sticky {
position: sticky;
top: 10px; /* 到达10px后粘住 */
}

5. Flexbox 布局

  • Flexbox 概述:

    Flexbox 是一种一维布局模型，用于在容器内分配元素的空间。

    设置 display: flex 创建一个 flex 容器，该容器的子元素称为 flex 项。

  • Flexbox 关键属性:
    • 容器属性:
      • flex-direction：定义主轴方向（行/列）。
      • justify-content：定义主轴上的元素排列方式（左对齐、右对齐、居中、两端对齐、均匀分布等）。
      • align-items：定义交叉轴上的元素排列方式（顶部对齐、底部对齐、居中、拉伸等）。
      • flex-wrap：定义子元素是否换行。
      • gap：定义子元素之间的间距。
    • 子元素属性:
      • flex-grow：定义元素的放大比例。
      • flex-shrink：定义元素的缩小比例。
      • flex-basis：定义元素的基础大小。
  • Flexbox 使用示例:
    • .container {
display: flex;
flex-direction: row; /* 水平布局 */
justify-content: space-between; /* 两端对齐 */
align-items: center; /* 垂直居中 */
flex-wrap: wrap; /* 换行 */
gap: 1rem; /* 间距 */
}

.item {
flex-grow: 1; /* 等比例放大 */
flex-basis: 200px; /* 基础宽度 */
}

6. 高级选择器

  • 子选择器 (>):

    仅选择直接子元素。

    例如：div > p { color: red; } 只选择 div 内的直接子元素 p。

  • 相邻兄弟选择器 (+):

    选择紧邻的兄弟元素。

    例如：h1 + p { margin-top: 0; } 选择紧跟在 h1 之后的 p。

  • 一般兄弟选择器 (~):

    选择同级的所有后续兄弟元素。

    例如：h1 ~ p { color: blue; } 选择 h1 后面的所有 p 元素。

7. 属性选择器

  • 属性选择器的用法:

    [attribute=value] 选择具有特定属性和值的元素。

    例如：input[type="number"] 选择所有类型为 number 的输入框。

    可以根据属性选择特定类型的元素，方便样式管理和组件开发。

  • 使用示例:
    • input[type="number"] {
background-color: #fed1ef; /* 特定类型的样式 */
}

8. 其他 CSS 技巧和注意事项

  • 消除文本装饰和列表样式:

    在一些默认情况下，链接元素（<a>）会带有下划线，列表（<ul> 和 <ol>）会带有默认的项目符号或编号。

    可以使用 text-decoration: none; 移除链接的下划线，使用 list-style: none; 移除列表的项目符号或编号。

    这些属性在定义全局样式时很常用，特别是在构建自定义导航菜单时。

    示例:

    a {
text-decoration: none; /* 去除链接的下划线 */
}

ul, ol {
list-style: none; /* 去除列表项的默认符号或编号 */
}
  • 禁止用户选择文本和拖拽元素:

    为了防止用户选中文本或拖拽图像，可以使用 user-select: none; 和 -webkit-user-drag: none;。

    user-select: none; 阻止用户选择文本，这在构建不可选中的按钮或自定义控件时非常有用。

    -webkit-user-drag: none; 特定于 WebKit 引擎（如 Chrome 和 Safari），可以防止图像被拖拽。

    示例:

    * {
user-select: none; /* 禁止选择文本 */
-webkit-user-drag: none; /* 禁止拖拽元素 */
}
  • 如何使用 calc() 函数:

    calc() 是一个 CSS 函数，用于动态计算长度值。它允许在 CSS 中使用加法、减法、乘法和除法来计算元素的尺寸。

    使用 calc() 可以实现响应式布局和动态尺寸计算。例如，可以使用 calc(100% - 50px) 来设置元素的宽度为容器宽度减去 50 像素。

    示例:

    .element {
width: calc(100% - 20px); /* 宽度是容器宽度减去20像素 */
height: calc(50vh - 10rem); /* 高度是视口高度的一半减去10rem */
}

    calc() 的优点:

    • 它可以在不使用 JavaScript 的情况下进行动态计算，适用于创建更灵活的布局。
    • 支持混合单位计算（例如：calc(100% - 50px)），这在处理相对和绝对尺寸时非常有用。
  • 如何理解 border-box 的使用场景:

    使用 box-sizing: border-box; 可以避免在设置宽度和高度时手动计算 padding 和 border 的尺寸。

    这种方法特别适合于需要精确控制元素尺寸的场景，如多列布局或弹性盒布局。

    例如，如果设置元素的宽度为 200px，并且设置了 padding 和 border，在默认的 content-box 模型下，实际元素的宽度会超过 200px。但是，在 border-box 模型下，宽度就被锁定为 200px，包括了 padding 和 border。

  • 定位元素脱离文档流的理解:

    在使用 absolute 或 fixed 定位时，元素会脱离文档流，这意味着它们不会影响其他元素的布局。

    absolute 定位相对于最近的已定位祖先元素（非 static 元素），如果没有已定位的祖先元素，则相对于 body。

    fixed 定位相对于视口，通常用于固定的导航栏或浮动广告。

    在脱离文档流时，要注意元素的堆叠顺序，可以通过 z-index 属性来控制。z-index 越大，元素就越在上层。

  • 调整 font-size 和使用 rem 单位的优势:

    设置 :root 的 font-size，例如 10px，可以让我们更直观地使用 rem 单位进行相对尺寸的定义。

    rem 单位基于根元素的 font-size，例如，如果 :root { font-size: 10px; }，那么 1rem 就等于 10px。这使得 rem 单位在响应式设计中非常实用。

    示例:

    :root {
font-size: 10px; /* 根元素字体大小 */
}

.element {
font-size: 1.6rem; /* 元素字体大小为16px（1.6 * 10px） */
}
  • 避免常见 CSS 错误和保持代码一致性:

    确保每个样式规则的结尾都使用分号 ;，即使它是规则的最后一条声明。这可以防止由于忘记添加分号而导致的语法错误。

    使用一致的代码格式和命名约定（如 BEM 命名法），以提高代码的可读性和可维护性。

    充分利用注释来解释复杂的样式规则或布局技巧，特别是在团队协作中，以帮助其他开发人员理解代码。
