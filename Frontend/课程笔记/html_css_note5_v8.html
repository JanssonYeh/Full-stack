Web Development Notes - Lesson 8

学习目标：

  • 掌握 Yarn 和 SCSS（Sass）的使用
  • 理解项目的基本结构和配置
  • 学习响应式设计和优化项目代码结构

步骤一：创建项目并安装依赖

1. 使用 Yarn 创建项目

yarn create vite

根据提示选择项目模板，例如：选择 "vanilla"（纯 JavaScript 项目），然后输入项目名称（小写）。

2. 安装 Sass

yarn add sass --dev

使用 --dev 参数将 Sass 标记为开发依赖（Development Dependency）。

步骤二：配置 Vite 和 SCSS

1. 配置 Vite

import { defineConfig } from 'vite';

export default defineConfig({
  // 配置选项
});

2. 使用 SCSS 编写样式

项目结构：

my-project/
│
├── index.html
├── src/
│   ├── main.js
│   └── styles/
│       └── main.scss
└── vite.config.js

index.html 文件：

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Vite and Sass Example</title>
</head>
<body>
  <div id="app">
    <h1>Hello, Vite and Sass!</h1>
    <button id="counter">Count: 0</button>
  </div>
  <script type="module" src="/src/main.js"></script>
</body>
</html>

main.scss 文件：

// 定义变量
$primary-color: #3498db;
$secondary-color: #2ecc71;
$text-color: #333;

// 使用嵌套规则和变量
body {
  font-family: Arial, sans-serif;
  background-color: $primary-color;
  color: $text-color;

  // 嵌套规则
  .header {
    background-color: $secondary-color;
    padding: 1rem;
    text-align: center;
  }

  .button {
    background-color: $primary-color;
    color: white;
    border: none;
    padding: 0.5rem 1rem;
    cursor: pointer;

    &:hover {
      background-color: darken($primary-color, 10%);
    }
  }
}

main.js 文件：

import './styles/main.scss';

步骤三：SCSS 的高级用法

1. 使用变量

// 定义变量
$primary-color: #3498db;
$secondary-color: #2ecc71;
$text-color: #333;

2. 嵌套规则

body {
  font-family: Arial, sans-serif;
  background-color: $primary-color;
  color: $text-color;

  // 嵌套规则
  .header {
    background-color: $secondary-color;
    padding: 1rem;
    text-align: center;
  }
}

3. 使用 Mixin

// 定义 Mixin
@mixin button-style($bg-color: $primary-color, $text-color: white) {
  background-color: $bg-color;
  color: $text-color;
  border: none;
  padding: 0.5rem 1rem;
  cursor: pointer;

  &:hover {
    background-color: darken($bg-color, 10%);
  }
}

// 使用 Mixin
.button {
  @include button-style;
}

步骤四：实现响应式设计

// 响应式设计示例
@media (max-width: 768px) {
  body {
    font-size: 14px;
  }

  .header {
    padding: 0.5rem;
  }
}

@media (min-width: 769px) {
  body {
    font-size: 16px;
  }

  .header {
    padding: 1rem;
  }
}

步骤五：版本控制与项目发布

1. 使用 Git 进行版本控制

git init

2. 创建 .gitignore 文件

# .gitignore 文件内容
node_modules/
dist/

3. 将项目提交到 GitHub

git add .
git commit -m "Initial commit"
git remote add origin <your-repo-url>
git push -u origin master

步骤六：项目代码结构和优化

优化后的项目结构：

my-project/
│
├── index.html
├── src/
│   ├── main.js
│   └── styles/
│       ├── _variables.scss
│       ├── _mixins.scss
│       └── main.scss
└── vite.config.js

main.scss 文件：

@import 'variables';
@import 'mixins';

body {
  font-family: Arial, sans-serif;
  background-color: $primary-color;
  color: $text-color;
}

额外说明

在使用这些工具和技术时，请确保对其有足够的理解。以下是一些额外的注意事项和技巧：

  • SCSS 变量： 变量应在项目中保持一致，避免重复定义相同的变量。
  • 响应式设计： 确保所有的媒体查询都能有效适配不同的设备，建议使用常见的断点。
  • Git 使用： 养成良好的提交习惯，每次提交前确保代码可以正常运行，并写清楚提交信息。
